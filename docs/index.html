File-Based P2P Encrypted Chatrooms

Inside every file is a chatroom trying to get out. Keykammer creates secure chatrooms using any arbitrary file as both the room identifier and encryption key. Users with the same file can connect to a private encrypted chat from anywhere on the internet.

WARNING

This is experimental software and needs testing. It probably won't harm your computer but is not guaranteed to work reliably. I'm publishing this website now to allow people to help with testing and contribute code to get this to a more polished and reliable state.

How It Works

1. Choose Any File

Use any file as your room key - a photo, document, or any arbitrary file becomes your private chatroom identifier.

2. Auto-Connect

Users with the same file automatically find each other through the discovery server and connect directly.

3. Private Chat

Messages are encrypted end-to-end using AES-256-GCM. No chat history is ever saved to disk.

Key Features

  • End-to-end encryption - AES-256-GCM with HKDF key derivation
  • Zero logging - No room data or messages stored anywhere
  • Ephemeral rooms - Rooms exist only while users are connected
  • Terminal interface - Clean TUI with real-time updates
  • Auto port forwarding - UPnP support for easy networking
  • Cross-platform - Works on Linux, macOS, and Windows
  • Open Source - View the code on GitHub

Quick Start

# Basic usage - any file becomes a chatroom
./keykammer photo.jpg

# With custom room size
./keykammer document.pdf -size 5

# Direct connection (no discovery server)
./keykammer myfile.txt -connect 192.168.1.100:53952
